Privacy & Security Privacy & Security Overview

This Mobile Inventory Management System helps view, add, update and delete inventory records. This page explains how the app handles protect privacy and security.

Data We Store

This app stores inventory data only, including:

  • Item ID (auto-generated)
  • Item name
  • Category (Electronics, Furniture, Clothing, Tools, Miscellaneous)
  • Quantity and price
  • Supplier name
  • Stock status (In Stock, Low Stock, Out of Stock)
  • Featured item flag
  • Special note (optional)

The app does not collect customer details, payment information, or other sensitive personal data.

Public API Usage

This app connects to a public API to retrieve and update inventory records. The API is used for:

  • Listing all items
  • Searching for items by name
  • Adding new inventory items
  • Updating or deleting existing items

Only inventory fields required for these operations are sent to the public API. Because the API is public, this app is intended for non-sensitive.
